# Sistema de diseño

_Documento generado durante el proyecto. Se completa conforme el 
Diseñador registra decisiones en `.project/decisions/designer.md`._

## Áreas a documentar

- Inventario de variables CSS existentes en el tema base.
- Tokens CSS: paleta de color, tipografía, espaciado, breakpoints, z-index.
- Comportamiento del header en sus distintos estados (inicial, scroll 
  parcial, scroll profundo).
- Posición y comportamiento de la barra de búsqueda avanzada por estado.
- Especificación de componentes: banda institucional, menú principal, 
  barra de búsqueda, footer.
- Comportamiento responsive por breakpoint.
- Estados interactivos de componentes (default, hover, focus, active, 
  disabled).
